# 待办事项应用

一个现代化的待办事项应用程序，具有本地存储功能。采用原生 HTML、CSS 和 JavaScript 构建。

## 功能特性

✨ **核心功能**
- ➕ 添加新任务
- ✅ 标记任务完成
- 📝 编辑现有任务
- 🗑️ 删除任务
- 🎯 过滤任务（全部、进行中、已完成）

📊 **统计数据**
- 总任务数
- 进行中的任务数
- 已完成的任务数

💾 **本地存储**
- 所有任务自动保存到浏览器本地存储
- 数据在浏览器会话间持久化
- 无需服务器支持

🎨 **用户体验**
- 美观的渐变色UI设计
- 响应式设计（支持手机和桌面）
- 流畅的动画和过渡效果
- 双击任务即可编辑
- 🎯 拖拽排序任务
- 实时更新统计信息

## 使用方法

### 添加任务
1. 在输入框中输入你的任务
2. 点击"添加任务"按钮或按 Enter 键
3. 任务将出现在列表中

### 完成任务
- 点击任务旁边的复选框标记为已完成
- 已完成的任务会显示删除线

### 编辑任务
**方法一：点击编辑按钮**
1. 点击任务上的"编辑"按钮
2. 修改输入框中的文本
3. 按 Enter 或再次点击"编辑"保存

**方法二：双击任务**
1. 双击任务文本
2. 编辑文本
3. 按 Enter 或点击外部保存

### 删除任务
- 点击"删除"按钮移除任务

### 过滤任务
- 点击"全部"查看所有任务
- 点击"进行中"查看未完成的任务
- 点击"已完成"查看已完成的任务

### 清除已完成任务
- 点击"清除已完成"按钮一次性删除所有已完成的任务

### 重新排序任务
- 点击并拖拽任务进行排序
- 新的顺序将自动保存

## 技术细节

### 本地存储
- 待办事项存储在 `localStorage` 中，键为 `'todos'`
- 数据以 JSON 字符串形式存储
- 每次操作后自动同步

### 数据结构
每个待办事项对象包含：
```javascript
{
    id: 1234567890,           // 基于时间戳的唯一ID
    text: "任务描述",          // 任务文本
    completed: false,          // 完成状态
    createdAt: "2024-01-01..." // ISO 时间戳
}
```

### 浏览器兼容性
- 支持所有现代浏览器（Chrome、Firefox、Safari、Edge）
- 需要启用 JavaScript
- 需要本地存储支持

## 使用技巧

1. **键盘快捷键**
   - 在输入框中按 `Enter` 添加任务
   - 编辑时按 `Enter` 保存
   - 编辑时按 `Escape` 取消

2. **移动端友好**
   - 按钮和复选框易于触摸操作
   - 所有屏幕尺寸的响应式布局
   - 支持离线使用（数据存储在本地）

3. **数据备份**
   - 在浏览器控制台导出待办事项：
   ```javascript
   copy(localStorage.getItem('todos'))
   ```
